## 权限系统的脚手架工具。
该脚手架已废弃，请使用命令行版

### 使用说明

1. 修改resources下application.yml的数据库地址
2. 修改generator.properties的author及包路径
3. 启动项目 访问 localhost:8777/index.html
4. 选择要导出的数据库表，浏览器会自动下载压缩包
